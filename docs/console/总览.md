# Console 功能总览

## 定位

OpenClaw 管理控制台，提供 Agent、Skill、Task、Session 的全生命周期管理及系统状态监控。

## 技术栈

| 类别 | 选型 |
|------|------|
| 框架 | React 19 |
| 语言 | TypeScript（strict mode） |
| 路由 | React Router 7 |
| 服务端状态 | TanStack Query 5 |
| 客户端状态 | Zustand 5 |
| 表单 | React Hook Form + Zod |
| 样式 | Tailwind CSS 4 |
| 组件库 | Base UI + 自建 |
| 图标 | Lucide React |
| 通知 | Sonner |
| 国际化 | react-i18next |
| 构建 | Vite 6 |

## 后端 API 接口清单

### Agents
| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/agents` | 获取所有 Agent |
| GET | `/api/agents/:id` | 获取单个 Agent |
| POST | `/api/agents` | 创建 Agent |
| PUT | `/api/agents/:id` | 更新 Agent |
| DELETE | `/api/agents/:id` | 删除 Agent |
| GET | `/api/agents/:id/soul` | 获取 Soul |
| PUT | `/api/agents/:id/soul` | 更新 Soul |
| GET | `/api/agents/:id/logs` | 获取审计日志 |

## 实施优先级

### P0 — 基础可用
1. 公共组件：Toast 挂载、确认弹窗、Dialog、Drawer
2. 类型对齐（TaskStatus、Agent.role、Task.priority/tags、Session.metadata）
3. Hooks 层补全：use-skills、use-tasks、use-sessions
4. Agent 创建 / 删除
5. Agent Soul 编辑

### P1 — 核心功能
6. Agent 信息编辑
7. Task 创建 / 详情抽屉 / 状态流转
8. Skill 创建 / 编辑 / 删除
9. Session 关闭
10. Dashboard 统计卡片 + 活动流

### P2 — 体验增强
11. 搜索与筛选（Agents、Skills、Sessions）
12. Agent 同步（预览 → 确认 → 执行）
13. 表单组件库统一
14. 空状态组件统一
15. 状态徽章组件抽取
16. 时间相对格式化
