# 04 - 控制台前端（apps/console）

## 职责

OpenClaw 管理界面，与 openclaw 服务配套。提供网关状态监控、agent/skill 管理、会话查看、任务看板等功能。

│   └── ui/           #   shadcn/ui 组件
├── hooks/            # 自定义 Hooks
│   ├── useSSE.ts     #   SSE 事件订阅
│   └── ...

## 页面功能

### Dashboard（仪表盘）

- 网关连接状态（实时）
- Agent 数量 / 在线数
- Skill 数量
- 活跃会话数
- 最近任务列表

### Agents（Agent 管理）

- Agent 列表（卡片展示，含状态和来源标签）
- 新建 / 编辑 / 删除
- 同步按钮（预览 → 确认 → 执行）
- 点击进入详情页

### Agent Detail（Agent 详情）

- 基本信息展示与编辑
- Soul 内容编辑器（textarea，加载/保存）
- 审计日志时间线

### Skills（Skill 管理）

- Skill 列表
- 来源筛选
- 详情查看

### Sessions（会话监控）

- 会话列表
- 会话详情（消息记录）

### Tasks（任务看板）

- Kanban 视图（pending / assigned / running / done / failed）
- 任务详情
- 手动创建任务

### Settings（系统设置）

- 网关连接配置
- 同步策略配置
- 关于信息

## 实时更新

通过 SSE 订阅 openclaw 服务的事件流，自动刷新对应的 TanStack Query 缓存：

- `agent.*` 事件 → 刷新 agents query
- `task.*` 事件 → 刷新 tasks query
- `session.*` 事件 → 刷新 sessions query
- `connection` 事件 → 更新 Zustand 连接状态
