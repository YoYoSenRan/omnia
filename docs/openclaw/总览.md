# OpenClaw 对接服务 功能总览

## 定位

Omnia 体系的核心后端服务（`apps/openclaw`），所有 Agent 能力的唯一入口。负责：

- 连接 OpenClaw Gateway，接入外部 Agent 生态
- 多源发现与同步 Agent（网关 / 本地磁盘 / 配置文件）
- 提供 REST API 供 Console 前端和业务服务（server）调用
- 维护 Agent、Skill、Task、Session 全生命周期
- 通过 SSE 推送实时事件

## 架构分层

```
HTTP 请求
  ↓
routes/       路由层 — 路径绑定、鉴权中间件选择
  ↓
controllers/  控制器层 — 参数提取、校验、响应格式化
  ↓
services/     服务层 — 业务逻辑、状态流转、事件触发
  ↓
db/repo/      数据访问层 — Drizzle 查询
  ↓
db/schema/    Schema 定义 — 表结构、类型推导
```

## 业务状态码

编码规则：5 位数字 XXYYZ（XX=模块，YY=错误序号，Z=保留）

| 模块 | 前缀 | 状态码 |
|------|------|--------|
| 通用 | 00 | SUCCESS(0) / BAD_REQUEST(400) / UNAUTHORIZED(401) / FORBIDDEN(403) / NOT_FOUND(404) / INTERNAL_ERROR(500) |
| Agent | 10 | AGENT_NOT_FOUND(10001) / AGENT_NAME_DUPLICATE(10002) / AGENT_SYNC_FAILED(10003) / AGENT_STATUS_INVALID(10004) / AGENT_SOUL_READ_FAILED(10005) / AGENT_SOUL_WRITE_FAILED(10006) |
| Skill | 20 | SKILL_NOT_FOUND(20001) / SKILL_SYNC_FAILED(20002) |
| Task | 30 | TASK_NOT_FOUND(30001) / TASK_STATUS_INVALID(30002) / TASK_ASSIGN_FAILED(30003) / TASK_TIMEOUT(30004) |
| Session | 40 | SESSION_NOT_FOUND(40001) / SESSION_CLOSED(40002) |
| Gateway | 50 | GATEWAY_CONNECT_FAILED(50001) / GATEWAY_DISCONNECTED(50002) / GATEWAY_AUTH_FAILED(50003) / GATEWAY_TIMEOUT(50004) |

## 启动流程

```
bootstrap()
  ↓
1. checkConnection()          — 数据库健康检查
  ↓
2. serve(app, port: 3301)    — 启动 HTTP
  ↓
3. GatewayClient.connect()   — 连接网关（GATEWAY_URL 非空时）
  ↓
4. handleGatewayFrame()      — 注册网关消息处理
  ↓
SIGTERM/SIGINT → 优雅关闭（HTTP → Gateway → DB Pool）
```

## 现状与规划

### 已实现

- 完整的 CRUD API（Agent / Skill / Task / Session）
- 网关 WebSocket 客户端（v3 协议、自动重连、心跳、RPC）
- 多源 Agent 同步（网关 / 磁盘 / 配置文件）
- Task 状态机（7 状态 + 合法流转校验）
- 事件总线 + SSE 推送（8 种事件）
- 两层鉴权（API Key + Service Token）
- 审计日志自动记录
- BaseService 泛型基类

### 待实现

| 优先级 | 模块 | 说明 |
|--------|------|------|
| P0 | 任务调度器（Scheduler） | 定时 tick 驱动后台任务 |
| P0 | 任务分派（Task Dispatch） | 自动将 assigned 任务分派给 Agent 执行 |
| P0 | Agent 执行协议 | 两阶段 gateway call → wait，拿到执行结果 |
| P1 | 任务队列端点 | Agent 主动拉取任务（GET /api/tasks/queue） |
| P1 | Skill 同步 | 从网关/磁盘同步 Skill 数据 |
| P2 | 多网关管理 | 支持连接多个 Gateway 实例 |
| P2 | Agent 间通信 | A2A 消息传递 |
