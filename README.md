# Omnia

OpenClaw 的 Web 控制台。管理 Agent、Skill、Workspace、Session，编排多 Agent 项目，追踪执行过程。

## 技术栈

- **前端**：Vite + React + TypeScript + shadcn/ui + Tailwind CSS
- **服务端**：Fastify + TypeScript + Drizzle ORM + SQLite
- **对接**：WebSocket → OpenClaw Gateway (Protocol v3)
